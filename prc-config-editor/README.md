# PRC流程配置编辑器

一个专为PRC流程配置文件设计的VS Code插件，提供智能补全、实时校验和语法高亮功能。

## 功能特性

### 🚀 智能代码补全
- **节点类型补全**：在第二个逗号位置自动提示节点类型
- **JSON结构补全**：为通讯类型节点自动插入JSON花括号模板
- **上下文感知**：根据当前位置智能提供相关补全建议
- **模糊匹配**：支持根据已输入内容过滤补全项

### ✅ 实时语法校验
- **列数验证**：确保每行包含足够的参数
- **节点类型验证**：检查第三列是否为有效节点类型
- **参数个数验证**：根据不同节点类型验证参数数量
- **数据格式验证**：
  - JSON格式校验
  - 非空字段验证
- **实时反馈**：在编辑过程中即时显示错误信息

### 🎨 语义语法高亮

- **基于列的高亮**：根据节点类型和列索引应用不同颜色
- **多种颜色分类**：包括注释内容、节点名称、节点说明、节点类型等
- **智能默认值**：为未定义列提供默认颜色

## 安装方法

### 手动安装
1. 获取 `.vsix` 文件
2. 在VS Code中按下快捷键 `Ctrl+Shift+P`
3. 选择 `Extensions: Install from VSIX...`
3. 选择`.vsix` 文件进行安装

## 使用方法

### 基本格式
PRC文件采用逗号分隔的格式，每行代表一个配置节点：节点名称,节点说明,节点类型,参数1,参数2,...

### 示例配置

```prc
// 写入数据示例
温度设定,设置加热器温度,写入数据,加热器1,{"Type":3,"StartAddress":100,"Length":1,"Value":150}

// 读取数据示例
温度读取,读取当前温度,读取数据,温度传感器,{"Type":3,"StartAddress":200,"Length":1}

// 执行函数示例
数据转换,转换温度单位,执行函数,TemperatureConversion,{"unit":"celsius"}

// 调用脚本示例
数据分析,分析历史数据,调用脚本,data_processor,analyze_trend
```

## 配置选项

在 VS Code 设置中自定义插件行为：

### 自动插入文件头 (Auto Insert Header)

- **设置路径**: `PRC配置编辑器` → `Prc Editor: Auto Insert Header`
- **默认值**: `true` (启用)
- **描述**: 控制新建 PRC 文件时是否自动插入包含节点配置说明的文件头
